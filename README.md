# Agile & DevOps Classwork

A collection of coursework and practical exercises focused on understanding Agile development practices and implementing DevOps automation using GitHub Actions. This repository primarily explores creating workflow automation using YAML configuration files and building Continuous Integration (CI) and Continuous Delivery (CD) pipelines.

---

## Overview

This repository was created as part of an Agile and DevOps course to gain hands-on experience with workflow automation and modern software development practices. The primary objective was to understand how GitHub Actions can automate repetitive development tasks such as building, testing, and deploying applications through workflow configuration files.

The exercises demonstrate how YAML-based workflows can be used to define automated pipelines that execute on specific GitHub events, improving development efficiency and ensuring consistent software delivery.

---

## Learning Objectives

- Understand Agile and DevOps fundamentals
- Learn GitHub Actions workflow automation
- Create and configure YAML (`.yml`) workflow files
- Automate build and testing processes
- Explore Continuous Integration (CI) concepts
- Understand Continuous Delivery (CD) pipelines
- Practice event-driven workflow execution
- Learn repository automation using GitHub Actions
